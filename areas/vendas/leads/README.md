# Vendas — Leads

Pasta para registro de leads, indicações e oportunidades comerciais.

## Padrão sugerido de nome
`AAAA-MM-DD-nome-ou-empresa.md`

## Campos mínimos
- Origem
- Status: novo / briefing / proposta / follow-up / ganho / perdido / pausado
- Dor principal
- Oferta discutida
- Próxima ação
- Data do próximo follow-up
