1. Escreva seu pedido em: \`[F2] memory/outputs/pedidos/SEU-NOME-PEDIDO-ASSUNTO.md\`

3. Seu arquivo deve conter: **quem solicita → para quem → o que precisa → prazo.**

Para: CCO
