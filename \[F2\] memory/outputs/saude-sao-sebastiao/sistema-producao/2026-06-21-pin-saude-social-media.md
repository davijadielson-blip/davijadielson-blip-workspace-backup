## Formato de entrega para copiar

Regra validada por Jadielson em 2026-06-21:

- ao entregar textos finais para stories, legendas, headlines ou peças rápidas, apresentar **todas as opções** de forma prática;
- usar opções numeradas, com título curto quando ajudar;
- cada opção deve vir em bloco limpo de código/texto, para facilitar copiar e colar;
- quando houver uma opção mais forte, marcar como **recomendada**, sem remover as demais;
- separar comentários/raciocínio do texto final copiável.

Modelo:

**Opção 1 — recomendada**

```text
Texto pronto para copiar.

Segunda linha, se houver.
```

**Opção 2 — variação**

```text
Outro texto pronto para copiar.
```
