# Grade Matriz — Grupos, Setores e Serviços da Saúde

Esta grade existe para evitar que a produção fique presa a campanhas ou aos mesmos setores. A lógica é:

**grupo fixo do dia → setor do grupo → serviço específico → tema do mês/dia → formato → CTA.**

Assim, cada campanha vira conteúdo local, e cada setor pode aparecer por vários ângulos.

---

## 1. Regra de uso da grade

Para montar uma pauta, seguir esta sequência:

1. Escolher o **grupo do dia**.
2. Escolher um **setor** dentro do grupo.
3. Escolher um **serviço/rotina** dentro do setor.
4. Cruzar com o **tema do mês, data de saúde ou necessidade da população**.
5. Definir o **formato**: stories, reels, carrossel, feed, card, cobertura.
6. Definir a função P.O.D.E:
   - Presença: mostrar que existe e está funcionando.
   - Originalidade: explicar como funciona e gerar valor.
   - Desejo: aproximar, humanizar, criar vínculo.
   - Engajamento: levar a uma ação clara.
7. Validar dados, fluxo de acesso, imagem e autorização.

## 2. Grade resumida por dia

| Dia | Grupo | Função principal | Tipo de pergunta que responde |
|---|---|---|---|
| Segunda | Atenção Básica / Território | Mostrar entrada da rede e presença perto de casa | Onde começa meu atendimento? |
| Terça | Serviços Especializados | Mostrar resolutividade e acesso a especialidades | Que serviço especializado existe e como acesso? |
| Quarta | Vigilância e Prevenção | Orientar, prevenir e antecipar riscos | Como evito problemas de saúde? |
| Quinta | Rede de Apoio e Humanização | Mostrar cuidado contínuo, vínculo e acolhimento | Quem cuida de casos que precisam de acompanhamento? |
| Sexta | +Flexível / Dia Extra | Abrir espaço para demandas operacionais, avisos de serviço, coberturas de oportunidade e pautas que não encaixem nos pilares fixos | O que precisa entrar hoje para orientar melhor a população? |

## 3. Grupo 1 — Atenção Básica / Território

**Dia recomendado:** segunda-feira  
**Papel editorial:** mostrar a saúde perto de casa, porta de entrada, rotina das unidades e vínculo com a comunidade.

### Setores e serviços possíveis

| Setor | Serviços/rotinas que podem virar pauta | Ângulos editoriais |
|---|---|---|
| UBS / PSF | acolhimento, recepção, triagem, consulta médica, enfermagem, acompanhamento de famílias | porta de entrada, cuidado contínuo, rotina da unidade |
| ACS | visitas domiciliares, busca ativa, orientação no território, acompanhamento de famílias | saúde que vai até a população, vínculo, prevenção |
| EAPs / atendimento estendido | atendimento em horário ampliado, organização de agenda, orientação à população | acesso, conveniência, ampliação do cuidado |
| Pré-natal | acompanhamento, orientação, consultas, vacinação da gestante | cuidado desde o começo, maternidade segura |
| Saúde da criança | puericultura, vacinação, acompanhamento de crescimento | prevenção, cuidado familiar |
| Crônicos | hipertensão, diabetes, acompanhamento regular | prevenção de complicações, rotina de cuidado |
| Recepção / marcação | orientação, direcionamento, organização do fluxo | como acessar, onde procurar, serviço certo |

### Conteúdos recorrentes

- UBS da semana.
- Agente de Saúde explica.
- Onde começa meu atendimento?
- O que levar para atendimento na unidade?
- Quando procurar a UBS?
- Como funciona o acompanhamento de hipertensão/diabetes?

### Temas de mês/dia que combinam

- vacinação;
- saúde da mulher;
- saúde do homem;
- diabetes;
- hipertensão;
- pré-natal;
- saúde da criança;
- campanhas sazonais.

## 8. Como mesclar campanha com grupo sem quebrar a rotação

A campanha não substitui o grupo. Ela entra como **filtro temático**.

Exemplo: campanha de vacinação

| Dia | Grupo | Como tratar vacinação |
|---|---|---|
| Segunda | Atenção Básica | sala de vacina na UBS, orientação na unidade |
| Terça | Especializados | vacinação de grupo específico atendido por serviço, se aplicável |
| Quarta | Vigilância | PNI, rede de frio, calendário vacinal, fake news |
| Quinta | Humanização | vacina em grupos vulneráveis, cuidado com idosos/crianças |
| Sexta | Bastidores | estoque, conservação, distribuição, balanço da campanha |

Exemplo: campanha de dengue/meio ambiente

| Dia | Grupo | Como tratar dengue/meio ambiente |
|---|---|---|
| Segunda | Atenção Básica | UBS orientando sintomas e quando procurar atendimento |
| Terça | Especializados | se houver atendimento/complicação/avaliação específica, usar com cuidado |
| Quarta | Vigilância | endemias, focos, orientação no território |
| Quinta | Humanização | cuidado com idosos, crianças e famílias em áreas de risco |
| Sexta | Bastidores | equipes em campo, planejamento, balanço, mapas/indicadores validados |

Exemplo: saúde mental / combate às drogas

| Dia | Grupo | Como tratar saúde mental |
|---|---|---|
| Segunda | Atenção Básica | porta de entrada, acolhimento inicial, orientação na unidade |
| Terça | Especializados | psicologia/psiquiatria, se fluxo estiver validado |
| Quarta | Vigilância/Prevenção | prevenção, sinais de alerta, informação segura |
| Quinta | Humanização | CAPS, família, oficinas, cuidado contínuo |
| Sexta | Bastidores | rede de apoio, articulação intersetorial, balanço de ações |

|---|---|---|---|---|---|
| Atenção Básica |  |  |  |  |  |

| Vigilância |  |  |  |  |  |
| Humanização |  |  |  |  |  |

**Regra prática:** repetir setor só quando houver motivo forte: campanha, serviço novo, alta demanda, correção de informação, resultado de ação ou necessidade de reforço público.

---

## 11. Crítica estratégica

A grade não deve virar prisão. Ela serve para garantir diversidade.

Se uma campanha muito forte cair na terça, por exemplo, não precisa ignorar. Mas o ideal é perguntar:

> Como essa campanha pode ser contada pelo grupo de terça?

Assim a campanha entra sem apagar a rotação da rede.
