# 🎬 SISTEMA DE TEMPLATES — Reels Otimizados
## Secretaria de Saúde São Sebastião

**Propósito:** Ter uma estrutura padrão pré-montada. Na hora de produzir, é só encaixar os novos takes e exportar.

---

## ⚙️ COMO FUNCIONA

```
TEMPLATE (estrutura fixa) + NOVOS TAKES (só troca o vídeo) = REEL PRONTO
```

Cada template tem:
- **Duração fixa** (15, 30 ou 45 seg)
- **Ritmo de edição pré-definido** (corte a cada X segundos)
- **Slots nomeados** pra encaixe de takes
- **Trilha + transições** já decididas
- **CTA fixo** no final

---

## 📦 TEMPLATE A — INFORMATIVO RÁPIDO (15 seg)

**Melhor para:** Dica rápida, aviso, campanha

### Estrutura (15 seg — 3 cortes)

```
┌─────────────────────────────────────────────┐
│  [ABERTURA]  Hook → 0-3 seg                 │
│  Tela limpa (cor sólida) + texto GRANDE     │
│  Ex: "VACINAÇÃO INFANTIL"                   │
├─────────────────────────────────────────────┤
│  [TAKE 1]  Explicação → 3-8 seg             │
│  Pessoa falando (plano médio, cenário UBS)  │
├─────────────────────────────────────────────┤
│  [TAKE 2]  Prova social → 8-12 seg           │
│  Imagem de apoio (cartão, atendimento real)  │
├─────────────────────────────────────────────┤
│  [CTA]  Chamada → 12-15 seg                 │
│  Textão: "Procure sua UBS"                  │
│  + Logo Secretaria                          │
└─────────────────────────────────────────────┘
```

**Trilha padrão:** Instrumental calmo, 80-90 BPM
**Transição:** Fade (0.3s) entre todos os blocos
**Legenda:** Sempre ativada (Captions automáticas)

### Checklist de produção
- [ ] Escrever hook (máx 4 palavras)
- [ ] Gravar take 1: pessoa fala 5 seg
- [ ] Capturar 1 imagem/video do serviço
- [ ] Juntar no template (arrastar takes pros slots)
- [ ] Exportar

## 📦 TEMPLATE B — PERSONAGEM + SERVIÇO (30 seg)

**Melhor para:** Profissional explicando, depoimento rápido

### Estrutura (30 seg — 5 cortes)

```
┌─────────────────────────────────────────────┐
│  [ABERTURA]  Hook visual → 0-4 seg          │
│  Pessoa já falando (não espera contar até 3) │
│  Ex: "Eu sou Maria, Agente de Saúde..."     │
├─────────────────────────────────────────────┤
│  [TAKE 1]  Contexto → 4-10 seg              │
│  Pessoa explica O QUE faz                    │
│  (plano médio, mãos visíveis)               │
├─────────────────────────────────────────────┤
│  [TAKE 2]  Onde faz → 10-17 seg             │
│  Corte pro local (andar pela UBS,          │
│  equipamento, sala de atendimento)          │
├─────────────────────────────────────────────┤
│  [TAKE 3]  Benefício → 17-24 seg            │
│  Volta pra pessoa + mostra resultado        │
│  Ex: "aqui a gente acompanha..."            │
├─────────────────────────────────────────────┤
│  [CTA]  Ação → 24-30 seg                    │
│  Tela final com endereço/horário + logo     │
│  Voz: "Vem pra UBS mais perto de casa"      │
└─────────────────────────────────────────────┘
```

**Trilha padrão:** Profissional, 90-100 BPM
**Transição:** Slide (0.4s) entre takes de local; Fade pros blocos de fala
**Efeito:** Leve zoom in no take 2 (movimento)

### Checklist de produção
- [ ] Roteiro de 30 seg ensaiado
- [ ] Gravar take 1 + 3 (pessoa falando, 2 ângulos)
- [ ] Gravar take 2 (B-roll do local, 7 seg de material)
- [ ] Capturar logo + endereço pro CTA
- [ ] Montar no template (encaixar takes + ajustar timing)

## 📦 TEMPLATE C — DADOS + IMPACTO (30 seg)

**Melhor para:** Prestação de contas, resultados, campanhas

### Estrutura (30 seg — 4 cortes)

```
┌─────────────────────────────────────────────┐
│  [ABERTURA]  Número impactante → 0-5 seg    │
│  Tela com número GRANDE + texto curto       │
│  Ex: "1.200 ATENDIMENTOS"                   │
│  Cor: Azul Claro (#0099CC) fundo           │
├─────────────────────────────────────────────┤
│  [TAKE 1]  Prova real → 5-13 seg            │
│  Pessoa explicando + imagens de ação        │
│  (plano americano, gestos naturais)         │
├─────────────────────────────────────────────┤
│  [TAKE 2]  Resultado visual → 13-22 seg     │
│  Compilado rápido de imagens do serviço     │
│  (3-4 imagens, 2 seg cada, transição fade)  │
├─────────────────────────────────────────────┤
│  [CTA]  Fechamento → 22-30 seg              │
│  Tela final: "Saúde é compromisso"          │
│  + Logo + Hashtag                           │
└─────────────────────────────────────────────┘
```

**Trilha padrão:** Inspirador/celebrativo, 105-115 BPM
**Transição:** Wipe nas transições de bloco
**Efeito:** Números animados (contador crescente) na abertura

### Checklist de produção
- [ ] Definir número principal (dado do período)
- [ ] Capturar 3-4 imagens de prova real
- [ ] Gravar take 1: gestor/profissional explica
- [ ] Selecionar trilha celebrativa
- [ ] Montar compilado de imagens + timing

## 📦 TEMPLATE D — BASTIDORES/HUMANIZAÇÃO (45 seg)

**Melhor para:** Sexta (bastidores), dia do profissional, CAPS

### Estrutura (45 seg — 6 cortes)

```
┌─────────────────────────────────────────────┐
│  [ABERTURA]  Cena real → 0-5 seg            │
│  Pessoa fazendo algo (não posando, REAL)    │
│  Ex: agente caminhando na rua               │
│  SOM ambiente natural (não corta)           │
├─────────────────────────────────────────────┤
│  [TAKE 1]  Quem é → 5-13 seg               │
│  Pessoa se apresenta + rotina               │
│  Plano médio, roupa de trabalho             │
├─────────────────────────────────────────────┤
│  [TAKE 2]  O trabalho → 13-22 seg           │
│  Mostrar a ação (atendimento, visita,       │
│  reunião) — plano detalhe                  │
├─────────────────────────────────────────────┤
│  [TAKE 3]  O humano → 22-30 seg             │
│  Pessoa fala do coração                     │
│  Ex: "o que me motiva é..."                 │
│  Close no rosto + olho no olho              │
├─────────────────────────────────────────────┤
│  [TAKE 4]  Contexto → 30-38 seg             │
│  Mostrar equipe/coletivo                    │
│  Plano aberto, todo mundo junto             │
├─────────────────────────────────────────────┤
│  [CTA]  Conexão → 38-45 seg                 │
│  Tela final: nome da pessoa + função        │
│  + "Conheça nosso time"                     │
│  Fundo: cor quente                          │
└─────────────────────────────────────────────┘
```

**Trilha padrão:** Emocional, 80-95 BPM (strings suaves)
**Transição:** Fade longo (0.5s) — tudo lento
**Efeito:** Nenhum efeito artificial — o realismo é o efeito

### Checklist de produção
- [ ] Escolher profissional do bem (boa comunicação)
- [ ] Preparar perguntas abertas (não roteiro fechado)
- [ ] Gravar B-roll da rotina (3-4 minutos de material)
- [ ] Capturar plano aberto da equipe
- [ ] Editar: cortar na emoção, não no timing exato

## 📦 TEMPLATE E — EMERGÊNCIA/URGÊNCIA (15 seg)

**Melhor para:** SAMU, alerta de campanha, prevenção sazonal

### Estrutura (15 seg — 3 cortes)

```
┌─────────────────────────────────────────────┐
│  [ABERTURA]  Alerta visual → 0-3 seg        │
│  Tela VERMELHA + texto "URGENTE"            │
│  + som de alerta (0.5seg)                   │
├─────────────────────────────────────────────┤
│  [TAKE 1]  O alerta → 3-9 seg               │
│  Profissional fala o risco                  │
│  Ex: "Dengue: 3 sinais de alerta"          │
│  Tom sério, direto                          │
├─────────────────────────────────────────────┤
│  [CTA]  Ação imediata → 9-15 seg            │
│  O que fazer + onde ir                      │
│  "Procure a UBS se tiver:" + lista          │
│  + Logo Secretaria                          │
└─────────────────────────────────────────────┘
```

**Trilha padrão:** Sem trilha (só voz + efeito)
**Transição:** Corte seco (sem transição)
**Cor fundo:** Vermelho Suave (#DD3333)

## 🗺️ MAPA DE USO — Qual template usar quando

| Situação | Template | Duração |
|----------|----------|---------|
| Dica rápida / aviso | **A** — Informativo | 15s |
| Profissional explicando serviço | **B** — Personagem | 30s |
| Prestação de contas / números | **C** — Dados | 30s |
| Bastidores / CAPS / humanização | **D** — Bastidores | 45s |
| Alerta / emergência / prevenção | **E** — Urgência | 15s |

## 🚀 FLUXO DE PRODUÇÃO COM TEMPLATES

```
        ROTEIRO                         PRODUÇÃO                     EDIÇÃO
 ┌──────────────────┐          ┌──────────────────┐         ┌─────────────────┐
 │ Escolher template│          │ Gravar takes      │         │ Abrir template   │
 │ (A/B/C/D/E)      │  ───►   │ seguindo checklist│  ───►   │ no editor        │
 │                  │          │ do template       │         │                  │
 │ Preencher        │          │                   │         │ Arrastar takes   │
 │ template branco  │          │ Dica: 2-3 takes   │         │ pros slots       │
 │                  │          │ por cena          │         │                  │
 │ Aprovar roteiro  │          │                   │         │ Ajustar cortes   │
 └──────────────────┘          └──────────────────┘         │ + legenda        │
                                                             │                  │
                                                             │ Exportar (H.264  │
                                                             │ 1080x1920 30fps) │
                                                             └─────────────────┘
```

## 💡 DICAS DE OTIMIZAÇÃO

1. **Grave takes genéricos** (abertura, explicação, CTA) em lote no mesmo dia — reutiliza em vários reels
2. **Mantenha uma pasta de ativos** no Google Drive com:
   - `audios/trilhas/` — trilhas já categorizadas por template
   - `assets/aberturas/` — aberturas animadas prontas
   - `assets/logos/` — logo em PNG (fundo transparente)
3. **Template mestre no editor** — deixe um projeto (Premiere/DaVinci/CapCut) com os cortes e transições já posicionados. Só substitui o clipe dentro de cada slot
4. **Batch rendering** — produza 5 reels de uma vez (mesmo template, takes diferentes)
