**Exemplo 1 — "Vale criar um serviço de YouTube?"**
```
LÔH convoca Conselho:

Lente Estratégia (COO+CMO+CAIO):
  → "Estamos em L2-Automated de vídeo; essa será nossa H2."

Lente Financeira (CFO):
  → "Setup custa R$5k, margem 40%, payback 3 meses."

Lente de Marca (CMO):
  → "Reforça posicionamento de 'produtora em crescimento'."

Lente Voz do Cliente (CRO):
  → "3 clientes já pediram; NPS ↑5 pontos se oferecer."

Lente Jurídica:
  → "Precisa checklist de direitos de imagem; nada proibitivo."

LÔH sintetiza:
  "Recomendo lançar: serve visão, afeta positivo marca,
   clientes pedem, financeiro OK. Risco baixo. Próximo passo:
   CCO desenha template + CAIO automatiza geração."

Jadielson: APROVA
```

```

Lente Jurídico: "CLT = compliance OK."
```
