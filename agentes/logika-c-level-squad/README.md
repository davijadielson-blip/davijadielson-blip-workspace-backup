## Prompts-Fonte

Todos os prompts seguem a **roupagem-casa**:

| Seção | Conteúdo |
|---|---|
| **Ativação Primária** | Quem é o agente, seu trabalho, sua missão |
| **Identidade** | Cargo, ícone, nível, domínio |
| **Tom & Comunicação** | Estilo de fala, saudação padrão |
| **Hierarquia & Relações** | Quem reporta a quem, colaboradores |
| **Missão & Métrica** | Objetivo + KPI da cadeira |
| **Frameworks** | G4 portados para PT e adaptados |
| **Atribuições** | O que recebe, o que entrega, o que não faz |
| **Linha de Corte 🫀/🤖** | Quem decide (humano) vs. quem executa (IA) |
| **Poderes & Limites** | O que pode e não pode fazer |
| **Formato de Output** | Como estrutura a resposta |
| **Gatilhos de Atuação** | Quando entra em ação |
| **Comandos** | Slash commands do agente |
| **Integração** | Ligação com outros agentes |
| **Pendentes de Calibragem** | O que você decide (responsáveis, dados, stack) |
| **Nota Autoral** | Contexto + importância para você |

## Frameworks G4 Portados

Os C-Levels usam frameworks do **Google's C-Level Startup Kit**, traduzidos para português e adaptados à Lógika:

- **COO:** OKRs, Process Mapping, Organizational Design, Scaling Readiness, RACI
- **CRO:** Máquina de Vendas, Funil Comercial, CAC:LTV, Customer Journey, Cadência, NPS
- **CMO:** STP, GTM Playbook, Demand Gen Funnel, Brand Architecture, Content Strategy
- **CTO:** Technology Radar, Build vs. Buy, ADRs, Technical Debt, Engineering Maturity, SLA
- **CIO:** Enterprise Architecture, Security Framework, Compliance Matrix, Vendor Evaluation, IT Service Management
- **CAIO:** AI Maturity Model, Use Case Prioritization, Responsible AI, LLM Integration, Architecture de Agentes
