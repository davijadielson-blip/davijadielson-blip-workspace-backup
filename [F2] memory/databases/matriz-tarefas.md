# Matriz dos 5 Destinos — Banco de Decisão por Tarefa

> Toda tarefa que entra na vida do Jadielson passa por aqui antes de virar compromisso.
> Atualizar sempre que nova tarefa for categorizada.

---

## Os 5 Destinos

| Símbolo | Categoria | Princípio |
|---|---|---|
| 🎯 **FOCO** | Só eu faço, no bloco Elite | Se eu não fizer, não fica como deveria |
| 🤝 **DELEGAR** | Sai da minha mão | Especificar quem assume e quando |
| ⚙️ **SISTEMA** | Vira template / checklist / processo | Se vai se repetir, precisa de padrão |
| 📦 **BLOCO** | Executar em janela fixa | Não faz picado — agrupa e despacha |
| 🗑️ **CORTAR** | Eliminar ou reduzir drasticamente | Não move ponteiro — para de fingir que move |

---

## Mini Filtro de 30 Segundos

> Quando aparecer tarefa nova, pergunte rápido:

```
1. Isso move o ponteiro?         → se sim, FOCO
2. Precisa ser eu especificamente? → se não, DELEGAR
3. Vai se repetir?               → se sim, SISTEMA
4. É pequena / secundária?       → BLOCO (janela fixa)
5. Não faz diferença real?       → CORTAR
```
