# gerar-follow-up-comercial

Use esta skill para reativar conversas comerciais sem pressão excessiva e com próximo passo objetivo.

## Entrada mínima
- Nome/contexto do lead
- Origem: indicação, inbound, cliente ativo, retorno etc.
- Último contato e status
- Oferta ou assunto tratado
- Objetivo do follow-up

## Saída padrão
- Mensagem curta para WhatsApp/DM
- Versão mais formal para e-mail/documento
- Próximo passo sugerido
- Nível de prioridade: quente/morno/frio

## Regra operacional
Não enviar mensagens automaticamente. Gerar rascunho para aprovação.
