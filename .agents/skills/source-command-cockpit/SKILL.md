# source-command-cockpit

Use this skill when the user asks to run the migrated source command `cockpit`.

## Command Template

Rode `bash scripts/cockpit/generate-cockpit.sh` para gerar o cockpit.html. Se estiver em ambiente interativo, o arquivo abre automaticamente no browser. Se quiser só visualizar sem regenerar, rode `open cockpit.html` (Mac) ou `start cockpit.html` (Windows).
