# organizar-contexto-comercial

Use esta skill para transformar materiais do banco natural no workspace em arquivos `.md` úteis para venda e produção. Notion, Drive, Obsidian ou GitHub não são fonte operacional primária; só entram mediante autorização/backup/consulta pontual.

## Pastas-alvo
- `areas/vendas/contexto/`
- `areas/marketing/contexto/`

## Saída padrão
- Resumo executivo
- Ofertas/serviços identificados
- Provas, cases e diferenciais
- Objeções e respostas
- Materiais de apoio
- Pendências de validação
- Fonte interna no workspace e data da consolidação

## Regra operacional
Separar fatos confirmados de inferências. Dados sensíveis devem ser resumidos sem exposição desnecessária.
