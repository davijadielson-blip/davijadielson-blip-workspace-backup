# MAPA — archive/

> O que vive nesta pasta: arquivos arquivados por contexto histórico. Não é lixeira.

## Estrutura

```text
archive/
└── MAPA.md          ← este arquivo
```

## Quando usar

- Materiais antigos do kit que não precisam ficar ativos.
- Versões substituídas de documentos importantes.
- Skills removidas, drafts publicados ou arquivos que podem ser úteis como referência.

## Quando NÃO usar

- Lixo ou arquivo quebrado sem valor → usar lixeira/trash quando disponível.
- Backups automáticos → ficam em `~/backups/`.
- Histórico de git → o git já cuida.
