## 🔧 Pipeline de Instruções (Jadielson → Lôh → Clara)

**Você aqui no chat:**
```
"Alô, avisa pra Clara reagendar — dia 10 em São José da Coroa Grande, cobertura mídia Saúde Movimento"
```

**Fluxo interno:**
1. Lôh recebe instrução
2. Lôh envia pra Clara (sessão privada)
3. Clara executa (agenda Google Calendar)
4. Clara reporta: "Feito, Lôh"
5. Lôh volta pra você: "Pronto, Clara reagendou dia 10"

**Você fica livre** — só me avisa qualquer coisa operacional
