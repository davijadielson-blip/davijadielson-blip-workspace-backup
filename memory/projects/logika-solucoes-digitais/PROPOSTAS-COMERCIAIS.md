# Propostas Comerciais — LÓGIKA | soluções digitais

> Objetivo: estruturar modelos comerciais vendáveis para a nova fase da LÓGIKA, evitando proposta genérica e facilitando fechamento.

## Marca nos documentos

**LÓGIKA | soluções digitais**  
**Conteúdo, tecnologia e crescimento digital.**

## Estrutura padrão de proposta

Toda proposta deve seguir esta estrutura:

1. **Capa**
   - Nome do cliente
   - Serviço proposto
   - Data
   - Marca LÓGIKA | soluções digitais

2. **Contexto do cliente**
   - O que entendemos do cenário
   - Principal dor/oportunidade
   - Objetivo da proposta

3. **Solução proposta**
   - O que a LÓGIKA vai fazer
   - Como será entregue
   - O que fica fora do escopo

4. **Entregáveis**
   - Lista objetiva do que será entregue
   - Quantidade, frequência e formato

5. **Processo de trabalho**
   - Briefing
   - Planejamento
   - Produção
   - Aprovação
   - Publicação/entrega
   - Relatório/ajustes

6. **Investimento**
   - Valor
   - Condições de pagamento
   - Validade da proposta

7. **Próximo passo**
   - Aprovar proposta
   - Enviar dados necessários
   - Agendar reunião de início

## Primeiros modelos prioritários

### 1. Social Media / Gestão de Conteúdo

Indicado para clientes que precisam de presença consistente nas redes.

**Possíveis entregáveis:**
- planejamento mensal;
- calendário editorial;
- criação de artes;
- legendas/copy;
- publicação ou entrega para publicação;
- acompanhamento básico;
- relatório simples.

### 2. Vídeo Institucional / Comercial

Indicado para empresas e instituições que precisam apresentar serviço, campanha, gestão ou produto.

**Possíveis entregáveis:**
- roteiro;
- captação;
- direção;
- edição;
- trilha/lettering;
- versão horizontal e/ou vertical;
- cortes para redes sociais, se contratado.

### 3. Pacote Presença Digital

Indicado para negócios que precisam organizar o básico: identidade de comunicação, redes, materiais e canais.

**Possíveis entregáveis:**
- diagnóstico de presença digital;
- bio/perfil;
- identidade verbal simples;
- templates de posts;
- calendário inicial;
- ajustes de WhatsApp/Instagram;
- recomendações de site ou landing page.

### 4. Site / Landing Page

Indicado para negócios que precisam de uma página clara para apresentar, captar leads ou vender.

**Possíveis entregáveis:**
- estrutura de página;
- copy;
- design;
- publicação;
- formulário de contato;
- integração básica com Notion/Make, quando aplicável.

### 5. Tráfego Pago

Indicado para clientes com oferta clara e verba de mídia.

**Possíveis entregáveis:**
- planejamento de campanha;
- configuração de campanha;
- criativos/copy;
- acompanhamento;
- relatório;
- ajustes de performance.

### 6. Automação e IA — fase inicial

Indicado para clientes com processos repetitivos simples.

**Importante:** vender inicialmente automações objetivas, não transformação digital total.

**Possíveis entregáveis:**
- diagnóstico de processo;
- desenho do fluxo;
- automação simples com Make.com;
- integração com formulário, planilha, Notion, e-mail ou WhatsApp, conforme viabilidade;
- teste e documentação básica.

## Regra comercial sobre automação/IA

Antes de vender automação, sempre responder:

1. Qual processo será automatizado?
2. Qual gatilho inicia o fluxo?
3. Onde os dados entram?
4. Onde os dados devem ser salvos?
5. Quem aprova/valida?
6. O que acontece se a automação falhar?
