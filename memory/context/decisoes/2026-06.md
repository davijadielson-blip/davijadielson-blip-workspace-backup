Jadielson determinou que todos os agentes, subagentes e tópicos devem seguir o mesmo caminho operacional:

- consultar obrigatoriamente o Cofre antes de responder;
- seguir o mapa geral do workspace para salvar;

A regra foi registrada em `MEMORY.md`, reforçada em `AGENTS.md` e propagada para prompts de agentes/tópicos na configuração OpenClaw.

## 2026-06-27 — Backup incremental condicional do Cofre

Jadielson pediu uma proteção em intervalo curto para não depender apenas do backup diário da madrugada.

Implementado:

- Script: `/data/.openclaw/workspace/scripts/backup-github-if-changed.sh`
- Frequência: a cada 30 minutos
- Cron: `backup-workspace-github-incremental-30min`
- Comportamento: faz `fetch` + `merge --ff-only`, verifica alterações reais, e só então `git add`, `commit` e `push` para `origin/main`.
- Não faz commit vazio; se não houver alterações, registra “Sem alterações; backup incremental ignorado.”
- Remote operacional permanece apenas `origin` (`davijadielson-blip-workspace-backup`). O remoto `backup` para `segundo-cerebro-jadielson` foi removido para evitar confusão.

Objetivo: reduzir risco de perda entre uma sessão de trabalho e o backup diário das 03:00 BRT.
