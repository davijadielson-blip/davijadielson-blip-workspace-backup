# MAPA — memory/

> O que vive nesta pasta: tudo que a Lôh precisa lembrar entre sessões de forma organizada.

## Estrutura

```text
memory/
├── MAPA.md          ← este arquivo
├── decisoes/        ← decisões importantes, preferencialmente por mês
└── projects/        ← projetos ativos, clientes, campanhas e iniciativas
```

## Convenção de nomes

- `decisoes/{YYYY-MM}.md` — decisões cronológicas do mês.
- `projects/{nome-curto}.md` — um arquivo por projeto/cliente/iniciativa.

## Quando usar

Use `memory/decisoes/` para registrar escolhas importantes: posicionamento da agência, preços, processos, ferramentas, preferências de comunicação, estratégia de conteúdo e combinados com clientes.

Use `memory/projects/` para acompanhar iniciativas ativas: clientes, campanhas, produção de vídeos, organização da agência, plano de marketing digital e transição de carreira.

## Atualizar este mapa quando

- Criar `pendencias.md`.
- Criar `people/` para contatos recorrentes.
- Criar `business/` para estruturar a agência.
- Criar subpastas por cliente ou área.
