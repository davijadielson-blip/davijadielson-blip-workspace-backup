# Mapa — [F2] memory/outputs/

## O que mora aqui
Todos os drafts gerados pela IA — legendas, roteiros, briefings, posts, resumos WhatsApp. Nada aqui vai direto para publicação: Jadielson revisa e marca `revisado: true`.

## O que NÃO mora aqui
Notas autorais (→ `[F1]`), contexto operacional (→ `[F2] memory/context/`), banco de dados permanente (→ `[F2] memory/databases/`).

## Subpastas
- `legendas/` — legendas Instagram por frente (`legendas/<frente>/`)
- `roteiros/` — roteiros de vídeo por frente
- `briefings/` — briefings de captação e produção
- `resumos-whatsapp/` — manchetes e resumos para distribuição
- `pessoal/` — outputs do @pessoal (parede-d'água)

## Convenções
- **Frontmatter obrigatório:** `revisado: false` ao criar; mudar para `true` após aprovação
- **Naming:** `YYYY-MM-DD-frente-slug.md`
- **Ciclo de vida:** draft criado → Jadielson revisa → `revisado: true` → publicado externamente → pode ser arquivado
