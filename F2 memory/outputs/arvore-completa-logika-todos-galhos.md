└─ fornece "voz do cliente" pra CMO

├─ monitora todos → custos por C-level

└─ fornece orçamento → restringe scope

CCO (Criação)
├─ recebe direção do CMO → executa conceitual
├─ trabalha com CAIO → templates de produção
└─ trabalha com Ewander → design

CAIO (IA/Automação)
├─ treina todos → cria seus agentes operacionais
├─ projeta automação com COO
└─ sugere oportunidades → "esse processo é L2-ready"
