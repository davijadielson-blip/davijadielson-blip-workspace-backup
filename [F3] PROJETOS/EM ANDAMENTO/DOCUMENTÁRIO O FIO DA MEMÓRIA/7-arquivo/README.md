# Cofre local — O FIO DA MEMÓRIA

Este é o cofre local do projeto no workspace principal do OpenClaw.

Uso previsto:
- guardar sínteses das conversas com Jadielson;
- registrar pesquisas complementares;
- preservar decisões criativas e operacionais;
- preparar versões de argumento, roteiro e plano de captação antes de irem para o Drive.

## Estrutura

- `01_pesquisas/` — notas de pesquisa, referências, achados históricos e conceituais.
- `02_argumento_roteiro/` — argumento, estrutura narrativa, roteiro de captação e versões.
- `03_decisoes/` — decisões aprovadas, pendências e alinhamentos.
- `04_producao/` — plano de filmagem, checklist, personagens, locações e organização prática.

## Regras

- O Drive fica como repositório dos arquivos do projeto.
- Este cofre local serve como memória de trabalho do agente.
- Nada deve ser apagado sem confirmação explícita de Jadielson.
