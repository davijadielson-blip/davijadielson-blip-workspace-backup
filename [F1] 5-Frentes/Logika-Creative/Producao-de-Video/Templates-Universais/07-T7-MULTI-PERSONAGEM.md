# 🗣️ T7 — MULTI-PERSONAGEM / DEBATE INTERNO (25s)

**Base viral:** Formato que MAIS CRESCE em 2026. Uma pessoa interpretando 3 papéis. Jump cuts a cada 3-5s = 72% mais chance de viralizar.

---

## 🎯 Quando usar
Produto/serviço com objeção de preço, decisão difícil, "comprar ou não", dúvida do cliente. Ideal para mostrar prós e contras de forma divertida.

---

## 📐 Estrutura de Timeline (25s — 6 cortes)

```
┌───────────────────────────────────────────────┐
│  [0-3s]  HOOK DE CONFLITO                    │
│  "O cético vs o entusiasta vs o realista"     │
│  Tela dividida ou mudança de persona rápida   │
├───────────────────────────────────────────────┤
│  [3-8s]  PERSONA 1 — O CÉTICO                │
│  "Ah, mas isso é caro/complicado/desnecessário"│
│  Tom duvidoso, expressão fechada              │
├───────────────────────────────────────────────┤
│  [8-13s] PERSONA 2 — O ENTUSIASTA             │
│  "Mas olha só o resultado! Vale cada centavo!"│
│  Tom animado, empolgado                       │
├───────────────────────────────────────────────┤
│  [13-18s] PERSONA 3 — O PRAGMÁTICO            │
│  "Na real, o que importa é o seguinte..."    │
│  Tom racional, dados na mão                   │
├───────────────────────────────────────────────┤
│  [18-22s] O VEREDITO (RESOLUÇÃO)              │
│  "No final das contas, a resposta é..."      │
│  Conclusão do debate                          │
├───────────────────────────────────────────────┤
│  [22-25s] CTA                                  │
│  "Qual persona você é? Comenta aqui"         │
│  + Logo                                       │
└───────────────────────────────────────────────┘
```

---

## 🅰️ Hooks que funcionam

| Ângulo | Hook exemplo | Gatilho |
|--------|-------------|---------|
| Conflito | "3 versões de mim discutindo se vale a pena" | Curiosidade |
| Identificação | "O cético, o ansioso e o decidido — qual é você?" | Identificação |
| Provocação | "Um lado diz SIM, o outro diz NÃO. Qual vence?" | Conflito |
| Interação | "A pessoa que acha caro vs a que já comprou" | Debate |
| Pergunta | "Seu [X] vs meu [X]: quem está certo?" | Engajamento |

## 🎬 Premiere — Montagem

### Timeline
```
V3:                [TEXTO DOS 3 LADOS]
V2:  [ TELA DIVIDIDA ]                   [VEREDITO][CTA]
V1:  [Hook] [Cético][Entusiasta][Pragmático][Resolução][CTA]
A1:  [PESSOA INTERPRETANDO OS 3 PAPÉIS]
A2:  [______TRILHA (dinâmica, leve, ritmada)______]
     0s   3s   8s     13s     18s      22s    25s
```

### Marcadores
| Timecode | Slot |
|----------|------|
| 00:00:00 | SLOT 1 — HOOK (0-3s) |
| 00:00:03 | SLOT 2 — CÉTICO (3-8s) |
| 00:00:08 | SLOT 3 — ENTUSIASTA (8-13s) |
| 00:00:13 | SLOT 4 — PRAGMÁTICO (13-18s) |
| 00:00:18 | SLOT 5 — VEREDITO (18-22s) |
| 00:00:22 | SLOT 6 — CTA (22-25s) |

### Técnica de edição
- **Jump cut** entre cada persona (corte a cada 3-5s)
- Cada persona: mude posição no quadro (esquerda/centro/direita)
- Mude expressão facial e tom de voz entre takes
- Texto na tela identificando cada persona

---

## 📱 CapCut — Montagem

### Timeline
```
V2:  [   "CÉTICO"   ]  [   "ENTUSIASTA"   ]  [   "PRAGMÁTICO"   ]
V1:  [HOOK] [CÉTICO] [ENTUSIASTA] [PRAGMÁTICO] [VEREDITO] [CTA]
A2:  [______TRILHA ANIMADA______]
```

### Legendas
- Nome da persona em destaque na tela
- Mudança de cor por persona (ex: vermelho/cético, verde/entusiasta, azul/pragmático)

### Transições
- Corte seco rápido (sem fade)
- Zoom sutil entre personas

---

## ✅ Checklist de produção

- [ ] 3 personas definidas (cético, entusiasta, pragmático)
- [ ] Roteiro por persona escrito
- [ ] Cada persona gravada em posição diferente no quadro
- [ ] Mudanças de expressão e tom entre takes
- [ ] Jump cuts aplicados (3-5s)
- [ ] Legendas com nome das personas
- [ ] Veredito/conclusão definido
- [ ] CTA interativo ("qual persona você é?")
- [ ] Logo no final
- [ ] Exportar: H.264, 1080×1920, 30fps

---

## 📊 Métricas de sucesso
- **Comentários:** MUITO alto (pessoas se identificam com uma persona)
- **Compartilhamentos:** Alto
- **Retenção:** >65% (jump cuts mantêm atenção)
