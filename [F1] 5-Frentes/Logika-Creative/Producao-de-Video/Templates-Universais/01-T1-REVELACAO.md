# 🎭 T1 — REVELAÇÃO / PLOT TWIST (15s)

**Base viral:** "Wait until the end" — 70% dos virais usam hook nos primeiros 2s

---

## 🎯 Quando usar
Institucional, resultado de serviço, surpresa, anúncio de novidade. Cria curiosidade e faz o espectador assistir até o final.

---

## 📐 Estrutura de Timeline (15s — 4 cortes)

```
┌──────────────────────────────────────────┐
│  [0-2s]  HOOK                          │
│  "Espera até ver..."                    │
│  Textão na tela ou pessoa falando       │
├──────────────────────────────────────────┤
│  [2-7s]  CONTEXTO / O PROBLEMA          │
│  Cena do "antes", situação atual         │
│  Pessoa explica o cenário                │
├──────────────────────────────────────────┤
│  [7-12s] REVELAÇÃO PARCIAL               │
│  "Mas aí aconteceu algo..."              │
│  Imagem de transição, suspense leve      │
├──────────────────────────────────────────┤
│  [12-15s] PLOT TWIST / RESULTADO         │
│  🔥 IMAGEM IMPACTANTE DO RESULTADO      │
│  + CTA: "Procure a [nome] e veja você"  │
│  + Logo                                  │
└──────────────────────────────────────────┘
```

---

- *Categoria Curiosidade:* abrir com pergunta que só o final responde
- *Categoria Choque:* afirmação surpreendente sobre o resultado

## 🎬 Premiere — Montagem

### Criar sequência
- **File → New → Sequence → Settings**
- Frame Size: **1080 × 1920** (9:16)
- Timebase: **29.97fps**
- Sample Rate: **48000 Hz**

### Marcadores de slot
| Timecode | Nome do slot |
|----------|-------------|
| 00:00:00 | SLOT 1 — HOOK (0-2s) |
| 00:00:02 | SLOT 2 — CONTEXTO (2-7s) |
| 00:00:07 | SLOT 3 — REVELAÇÃO (7-12s) |
| 00:00:12 | SLOT 4 — PLOT TWIST + CTA (12-15s) |

### Tracks
```
V3:  [                       ] [B-ROLL SURPRESA]
V2:  [       TEXTO HOOK       ]          [TELA FINAL]
V1:  [HOOK] [CONTEXTO] [REVELAÇÃO] [RESULTADO]
A1:  [LOCUÇÃO: explica o cenário]
A2:  [________TRILHA (crescente até o auge)________]
     0s     2s        7s        12s       15s
```

### Configuração de áudio
- Trilha: começa baixa, **aumenta gradualmente** até o plot twist
- Efeito sonoro no momento da revelação (whoosh/riser + thud)
- Locução: clara, tom de storyteller

## 📱 CapCut — Montagem

### Novo projeto
- Proporção: **9:16**
- Resolução: **1080×1920**
- FPS: **30fps**
- Duração: **15 segundos**

### Timeline
```
V2:  [  TEXTO: "ESPERA ATÉ VER"  ]         [CTA FINAL]
V1:  [TAKE A] [TAKE B] [TAKE C] [TAKE D]
A2:  [________TRILHA COM CRESCENDO________]
```

### Transições
- **Corte seco** entre HOOK → CONTEXTO
- **Zoom in sutil** na REVELAÇÃO (120% → 100%)
- **Flash branco** de 0.2s no PLOT TWIST

### Legendas
- Grandes, negrito
- Palavra-chave destacada em **AMARELO ou VERMELHO**
- Animação: aparecer letra por letra no hook
