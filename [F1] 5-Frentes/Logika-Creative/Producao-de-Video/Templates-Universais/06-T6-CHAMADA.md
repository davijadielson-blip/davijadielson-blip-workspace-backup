# 📢 T6 — CHAMADA / EVENTO (12s)

**Base viral:** Conteúdo de chamada funciona melhor quando é curto, direto e cria urgência visual. O CTA deve ser a estrela.

---

## 🎯 Quando usar
Convocação para evento, campanha sazonal, aviso importante, data comemorativa, lançamento. Qualquer conteúdo que precisa que a pessoa AJA.

---

## 📐 Estrutura de Timeline (12s — 3 cortes)

```
┌───────────────────────────────────────────────┐
│  [0-2s]  HOOK DE URGÊNCIA                    │
│  Visual impactante + data/texto chamativo     │
│  "É AGORA" / "NÃO PERCA" / "CHEGOU O DIA"    │
├───────────────────────────────────────────────┤
│  [2-7s]  O QUE É / INFORMAÇÃO                 │
│  O evento/serviço em ação                     │
│  "Dia X, na Y, a partir das Z"               │
│  Imagens do que a pessoa vai encontrar        │
├───────────────────────────────────────────────┤
│  [7-12s] CTA DIRETO                           │
│  Chamada forte para ação                      │
│  "Vai perder? Corre que ainda dá tempo"       │
│  + Logo + data/local (se aplicável)           │
└───────────────────────────────────────────────┘
```

---

## 🅰️ Hooks que funcionam

| Ângulo | Hook exemplo | Gatilho |
|--------|-------------|---------|
| Urgência | "FALTA POUCO. Não perca [evento]" | FOMO |
| Promessa | "No dia X, tudo que você precisa saber sobre [tema]" | Expectativa |
| Direto | "SE VOCÊ [perfil], ISSO É PRA VOCÊ" | Segmentação |
| Identificação | "Quem esperou o ano todo por [evento], chegou a hora" | Identificação |
| Número | "[X] dias. É o que falta para [evento] acontecer" | Contagem |

## 🎬 Premiere — Montagem

### Timeline
```
V3:         [B-ROLL DO EVENTO / DO SERVIÇO]
V2:  [ TEXTO: "É AGORA" ]  [TEXTO: INFO]     [CTA FORTE]
V1:  [HOOK] [O QUE É] [CHAMADA PRA AÇÃO]
A1:  [LOCUÇÃO ENÉRGICA]
A2:  [______TRILHA (crescendo até o CTA)______]
     0s   2s         7s           12s
```

### Marcadores
| Timecode | Slot |
|----------|------|
| 00:00:00 | SLOT 1 — HOOK (0-2s) |
| 00:00:02 | SLOT 2 — O QUE É (2-7s) |
| 00:00:07 | SLOT 3 — CTA (7-12s) |

### Edição
- Ritmo: **acelerado** (pouco tempo, cada frame importa)
- Cores: vibrantes, contraste alto
- Texto na tela: mínimo (só data, horário, local, ação)

---

## 📱 CapCut — Montagem

### Timeline
```
V2:  ["É AGORA"]  [INFO RÁPIDA]  ["VAI PERDER?"] 
V1:  [IMPACTO] [EVENTO/SERVIÇO EM AÇÃO] [CTA]
A2:  [______TRILHA CRESCENTE______]
```

### Legendas
- Curtas, grossas, letras grandes
- CTA final: ocupa a tela inteira

### Transições
- Corte seco rápido
- Flash ou zoom out no CTA

---

## ✅ Checklist de produção

- [ ] Data/hora/local/link confirmados
- [ ] Hook de urgência ou promessa definido
- [ ] Imagens do evento/serviço capturadas
- [ ] Texto na tela: mínimo necessário
- [ ] CTA forte definido (o que a pessoa precisa FAZER)
- [ ] Logo + dados do evento no final
- [ ] Exportar: H.264, 1080×1920, 30fps

---

## 📊 Métricas de sucesso
- **Compartilhamentos:** Alto (marcam amigos "vamos?")
- **Cliques/Curtidas:** Indicador de conversão
- **Retenção esperada:** >60%
