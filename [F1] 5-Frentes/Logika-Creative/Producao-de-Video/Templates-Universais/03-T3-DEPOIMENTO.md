# 💬 T3 — DEPOIMENTO / HISTÓRIA PESSOAL (25s)

**Base viral:** Histórias emocionais geram 25% mais compartilhamentos. Storytelling com obstáculo + resolução é o formato mais retido.

---

## 🎯 Quando usar
Cliente satisfeito, caso de sucesso, prova social, conexão emocional, depoimento de paciente/usuário. Gera identificação e confiança.

---

## 📐 Estrutura de Timeline (25s — 5 cortes)

```
┌───────────────────────────────────────────────┐
│  [0-3s]  HOOK DE IDENTIFICAÇÃO                │
│  "Se você já passou por isso..."              │
│  Pessoa já falando, expressão genuína          │
├───────────────────────────────────────────────┤
│  [3-9s]  O PROBLEMA / A DOR                   │
│  "A parte mais difícil foi..."                │
│  Pessoa descrevendo a situação antes           │
│  Empatia, tom sincero                          │
├───────────────────────────────────────────────┤
│  [9-14s] O PONTO DE VIRADA                    │
│  "Mas aí eu descobri/tentei/fiz..."            │
│  O momento que mudou (o serviço/produto)      │
├───────────────────────────────────────────────┤
│  [14-21s] O RESULTADO / A TRANSFORMAÇÃO        │
│  "Hoje minha vida é diferente..."              │
│  Imagem do resultado + pessoa satisfeita       │
├───────────────────────────────────────────────┤
│  [21-25s] CTA + ENCERRAMENTO                   │
│  "Se você também quer, procura..."            │
│  + Logo                                        │
└───────────────────────────────────────────────┘
```

---

## 🅰️ Hooks que funcionam

| Ângulo | Hook exemplo | Gatilho |
|--------|-------------|---------|
| Identificação | "Se você já sentiu que não ia conseguir..." | Identificação |
| História | "Essa é a parte mais difícil de contar..." | Empatia |
| Vulnerabilidade | "Eu nunca pensei que ia compartilhar isso..." | Confiança |
| Dilema | "Eu estava prestes a desistir quando..." | Suspense |
| Virada | "Foi só quando alguém me disse isso que tudo mudou" | Curiosidade |

**Hooks do vault para adaptar (prompt 510):**
- Ângulo de **experiência de cliente com resultado específico**
- Ângulo de **dilema identificável (empatia)**

## 🎬 Premiere — Montagem

### Timeline
```
V3:      [B-ROLL do resultado/antes]
V2:    [       TEXTO DE APOIO       ]       [CTA]
V1:  [Hook][Problema][Virada][Resultado][Encerramento]
A1:  [_________DEPOIMENTO (pessoa falando)________]
A2:  [______TRILHA (crescente até virada, suave depois)______]
     0s   3s       9s       14s      21s     25s
```

### Marcadores
| Timecode | Slot |
|----------|------|
| 00:00:00 | SLOT 1 — HOOK (0-3s) |
| 00:00:03 | SLOT 2 — PROBLEMA (3-9s) |
| 00:00:09 | SLOT 3 — VIRADA (9-14s) |
| 00:00:14 | SLOT 4 — RESULTADO (14-21s) |
| 00:00:21 | SLOT 5 — CTA (21-25s) |

### Técnica de edição
- Cortes: **jump cut** a cada 3-5s (remove pausas, mantém ritmo)
- Transições: fade de 0.2s entre blocos
- No RESULTADO: slow motion sutil no momento emocionante
- Música: começa baixa, sobe no resultado

## 📱 CapCut — Montagem

### Timeline
```
V2:  [TEXTO DO PROBLEMA]      [TEXTO RESULTADO]  [CTA]
V1:  [PESSOA FALANDO - takes alternados com B-roll]
A1:  [ÁUDIO DO DEPOIMENTO]
A2:  [________TRILHA EMOCIONAL________]
```

### Legendas
- Grandes, **jump sync** (palavra por palavra no ritmo da fala)
- Palavras emocionais destacadas em cor
- Evite bloquear o rosto da pessoa

### Transições
- **Cross dissolve** suave entre takes
- Zoom in gradual no RESULTADO (105%)

## ✅ Checklist de produção

- [ ] Pessoa escolhida (cliente real, tom genuíno)
- [ ] Roteiro de perguntas guiadas (não script decorado)
- [ ] Hook de identificação definido
- [ ] B-roll do "antes" e "depois" capturado
- [ ] Jump cuts aplicados (remover pausas)
- [ ] Trilha emocional escolhida (suave, lo-fi)
- [ ] Legendas revisadas (jump sync)
- [ ] CTA e logo no final
- [ ] Exportar: H.264, 1080×1920, 30fps
