# ⚡ T4 — DICA RÁPIDA / EDUCATIVO (15s)

**Base viral:** Conteúdo educativo com hook de "pare de fazer isso" gera altas taxas de save. "Nobody told me this" é um dos hooks mais usados globalmente.

---

## 🎯 Quando usar
Dica rápida, utilidade pública, "pare de fazer isso", mito vs verdade, curiosidade informativa. Ideal para Saúde, Câmara e institucional.

---

## 📐 Estrutura de Timeline (15s — 3 cortes)

```
┌───────────────────────────────────────────────┐
│  [0-3s]  HOOK PROVOCATIVO                    │
│  "Pare de fazer isso AGORA"                  │
│  Textão na tela + pessoa falando              │
├───────────────────────────────────────────────┤
│  [3-9s]  O ERRO / O MITO                     │
│  "A maioria das pessoas faz assim..."        │
│  Mostra o que NÃO fazer / o problema          │
├───────────────────────────────────────────────┤
│  [9-15s] A SOLUÇÃO / A DICA                  │
│  "Na verdade, o certo é..."                  │
│  🔥 Demonstração prática ou texto claro      │
│  + CTA: "Salva pra não esquecer"             │
│  + Logo                                       │
└───────────────────────────────────────────────┘
```

---

## 🅰️ Hooks que funcionam

| Ângulo | Hook exemplo | Gatilho |
|--------|-------------|---------|
| Provocação | "Pare de fazer isso se você quer resultado" | Atenção negativa |
| Dado chocante | "70% das pessoas fazem esse erro todo dia sem saber" | Dado |
| Utilidade | "Ninguém te contou esse truque simples" | Curiosidade |
| Mitologia | "O que sua mãe te ensinou sobre [tema] está ERRADO" | Controvérsia |
| Número | "3 segundos. É só o que você precisa para [resultado]" | Especificidade |

**Hooks do vault para adaptar (prompts 002, 510, 838):**
- Ângulo de **contra-intuição** (quebrar crença comum)
- Ângulo de **dado específico** (número ou fato surpreendente)

## 🎬 Premiere — Montagem

### Timeline
```
V3: [                         ][B-ROLL da dica]
V2: [   TEXTO: "PARE DE FAZER ISSO"   ]        [CTA]
V1: [HOOK] [O ERRO / O MITO] [SOLUÇÃO PRÁTICA]
A1: [LOCUÇÃO direta, didática]
A2: [______TRILHA LEVE (instrumental baixo)______]
     0s   3s           9s           15s
```

### Marcadores
| Timecode | Slot |
|----------|------|
| 00:00:00 | SLOT 1 — HOOK (0-3s) |
| 00:00:03 | SLOT 2 — ERRO/MITO (3-9s) |
| 00:00:09 | SLOT 3 — SOLUÇÃO + CTA (9-15s) |

### Edição
- Cortes: ritmo rápido, jump cuts a cada 2-3s
- Texto na tela: GRANDE, negrito, cor contrastante
- Zoom in na SOLUÇÃO para dar peso

## 📱 CapCut — Montagem

### Timeline
```
V2:  [   TEXTO GRANDE DO HOOK  ]      [DICA NA TELA]
V1:  [PESSOA FALANDO] [CENA DO ERRO] [DEMONSTRAÇÃO]
A2:  [_____TRILHA LEVE E CONSTANTE_____]
```

### Legendas
- Muito grandes, ocupando 60-70% da largura
- Palavra-chave em **DESTAQUE COLORIDO**
- Animação: bounce ou pop no hook

### Transições
- Corte seco (sem fade) pra manter ritmo acelerado
- Zoom out no mito, zoom in na solução

## ✅ Checklist de produção

- [ ] Gancho provocativo ou de utilidade definido
- [ ] Cena do "erro/mito" capturada (se aplicável)
- [ ] Demonstração da solução gravada
- [ ] Textos na tela preparados (solução em destaque)
- [ ] Ritmo acelerado (corte a cada 2-3s)
- [ ] CTA "Salva pra não esquecer"
- [ ] Logo no final
- [ ] Exportar: H.264, 1080×1920, 30fps

---

## 📊 Métricas de sucesso
- **Saves:** Altíssimo (formato mais salvo)
- **Compartilhamentos:** Alto (pessoas marcam outras)
- **Retenção:** Esperada >70%
