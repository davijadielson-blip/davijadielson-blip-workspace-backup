# 🔧 T5 — BASTIDORES / DIA A DIA (20s)

**Base viral:** 50% dos usuários preferem conteúdo autêntico sem produção. Câmera na mão, sem tripé, sem estúdio. Lo-fi content domina 2026.

---

## 🎯 Quando usar
Humanização da marca/instituição, mostrar equipe, making of, preparação, confiança. Conteúdo que mostra o "nos bastidores" do serviço.

---

## 📐 Estrutura de Timeline (20s — 4 cortes)

```
┌───────────────────────────────────────────────┐
│  [0-3s]  HOOK VISUAL + CONTEXTO              │
│  Pessoa já em ação, câmera na mão             │
│  "Vem comigo ver como é [ação]..."            │
│  Estilo vlog, natural                          │
├───────────────────────────────────────────────┤
│  [3-10s] A AÇÃO (O PROCESSO)                  │
│  Corte rápido de várias etapas do processo    │
│  "Primeiro a gente faz X, depois Y..."        │
│  Montagem dinâmica, takes curtos              │
├───────────────────────────────────────────────┤
│  [10-15s] O DIFERENCIAL / O "SEGREDO"         │
│  "O que ninguém vê é que..."                  │
│  Revela algo que o público não sabe            │
├───────────────────────────────────────────────┤
│  [15-20s] RESULTADO + CTA                     │
│  O trabalho finalizado                        │
│  "É por isso que a gente faz assim"          │
│  + Logo                                       │
└───────────────────────────────────────────────┘
```

---

## 🅰️ Hooks que funcionam

| Ângulo | Hook exemplo | Gatilho |
|--------|-------------|---------|
| História | "Vem comigo ver o que acontece antes de [serviço]" | Curiosidade |
| Identificação | "Todo mundo pergunta como a gente faz isso" | Identificação |
| Bastidores | "O que ninguém vê nos bastidores de [trabalho]" | FOMO |
| Autenticidade | "Sem filtro: como realmente é um dia de [profissão]" | Autenticidade |
| Revelação | "O segredo que faz nossa [serviço] ser diferente" | Curiosidade |

## 🎬 Premiere — Montagem

### Timeline
```
V3:                    [B-ROLLDIFERENCIAL]
V2:     [      TEXTOS DE APOIO      ]        [CTA]
V1:  [HOOK] [AÇÃO - MONTAGEM] [SEGREDO] [RESULTADO]
A1:  [LOCUÇÃO SOLTA, ESTILO VLOG]
A2:  [______TRILHA (dinâmica, ritmada)______]
     0s   3s          10s      15s      20s
```

### Marcadores
| Timecode | Slot |
|----------|------|
| 00:00:00 | SLOT 1 — HOOK (0-3s) |
| 00:00:03 | SLOT 2 — AÇÃO/MONTAGEM (3-10s) |
| 00:00:10 | SLOT 3 — DIFERENCIAL (10-15s) |
| 00:00:15 | SLOT 4 — RESULTADO + CTA (15-20s) |

### Estilo de edição
- **Câmera na mão** (estabilização leve, sem tripé)
- Cortes: montagem rápida na AÇÃO (vários takes de 1-2s)
- Cores: naturais, sem filtro pesado (authentic look)
- Música: dinâmica, upbeat (tipo lo-fi beats)

## 📱 CapCut — Montagem

### Timeline
```
V2:  [   TEXTOS CURTOS   ]       [TEXTO: "E É POR ISSO"]
V1:  [CENA 1][VÁRIOS TAKES RÁPIDOS] [REVELAÇÃO] [FINAL]
A2:  [______BEAT LO-FI______]
```

### Estilo
- Sem transições elaboradas (corte seco)
- Pode tremer um pouco a câmera (autenticidade)
- Legendas: no estilo "legenda de vlog"

### Transições
- Corte seco
- Match cut entre ações similares

---

## ✅ Checklist de produção

- [ ] Hook "vem comigo" ou bastidores definido
- [ ] Cenas do processo capturadas (mínimo 4 takes)
- [ ] Algo que "ninguém vê" identificado
- [ ] Estilo câmera na mão (sem produzir demais)
- [ ] Música lo-fi escolhida
- [ ] Cena final do resultado capturada
- [ ] Logo no final
- [ ] Exportar: H.264, 1080×1920, 30fps

---

## 📊 Métricas de sucesso
- **Confiança:** Muito alto (autenticidade constrói credibilidade)
- **Comentários:** Alto (pessoas se identificam)
- **Retenção:** >60%
