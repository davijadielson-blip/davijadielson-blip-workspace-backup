# LÓGIKA CODE — REBRANDING

> Avaliar e definir um novo nome/posicionamento para a frente de tecnologia da Lógika — separando comunicação/audiovisual de automação/sistemas.

---

## 🧭 Por que existe

A Lógika Creative/Films tem posicionamento claro em audiovisual. Com a expansão para automações, apps e sistemas (WhatsApp bots, etc.), surgiu a necessidade de uma marca ou frente separada — para não diluir a identidade nem confundir o mercado.

---

## 🎯 Resultado esperado

Nome definitivo escolhido, proposta de valor escrita em 1 frase, e decisão sobre se será sub-marca da Lógika ou marca independente.

---

## 📚 Precisa estudar / pesquisar antes

> Nenhuma fase avança sem responder isto aqui primeiro.

- [ ] Qual é o público-alvo desta frente tech? (PMEs? comunicadores? setor público?)
- [ ] O nome "Lógika Code" já está disponível (domínio, Instagram, CNPJ)?
- [ ] Faz sentido criar CNPJ separado ou manter como frente da Lógika?
- [ ] Concorrentes diretos com posicionamento similar — como se diferenciar?
